# HE Universal Data Prep Agent

This repository provides a lightweight agent workflow for health economics (HE) model parameter evidence preparation.

## Current scope
- **Phase 1 only**: task intake, rationale document conversion, publication search/selection, and Excel export.
- No UI, no HE model calculations, and no Phase 2 features.

## Where to start
See [`phase_1/README.md`](phase_1/README.md) for setup, run instructions, and examples.
