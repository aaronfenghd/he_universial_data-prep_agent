# HE Universal Data Prep Agent

This repository provides a lightweight agent workflow for health economics (HE) model parameter evidence preparation.

## Current scope
- **Phase 1**: task intake, rationale document conversion, publication search/selection, and Excel export.
- **Phase 2**: a lightweight Streamlit chat app that orchestrates the Phase 1 workflow.
